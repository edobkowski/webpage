NeuroPredictor

Language: Python

What have I learned: basic aspects of deeplearning, recurental neural networks, estimation of time series, python libraries: pandas, numpy, keras, sklearn

Overwiev: The aim of the project was to design a system based on the recurrent neural network architecture, capable of learning on the provided data set by processing signals in subsequent layers of elements (artificial neurons). Applications of the program may be comprehensive, however, limited to predicting future values of only such variables, which are characterized by a certain pattern. Due to this program can be used for all kinds of forecasts, especially financial time series (forecasts of sales prices, stock market forecasts). An example of using the program may be a stock price forecast based on historical data from the Yahoo! Finance.
