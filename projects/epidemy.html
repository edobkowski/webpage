Epidemy

Language: C++

What have I learned: Object Oriented Programming, shortest path finding algorithms (Djikstra, Floyd-Warshall), cellural automata systems

Overwiev: This is one of my favourites projects which gave me a lot of fun while I have been working on it. I wrote this program as my bachelor diploma on subject Modeling epidemics of selected infectious diseases using cellular automata.

A purpose of this work was to build a cellular-automata based computer program to simulate epidemic spread and evolution of infectious diseases under specific conditions. The model was constructed to be universal as much as possible and to enable the user to enter his own set of parameters. To check an influence of communications tracks existence on epidemic spread I have implemented Floyd-Warshall algorithm and experimented with other shortest path finding algorithms.
